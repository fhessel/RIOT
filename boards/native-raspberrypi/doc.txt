@ingroup     boards
@brief       Support for running RIOT in native with physical GPIO support

# Overview
![Terminal running RIOT native](https://raw.githubusercontent.com/wiki/RIOT-
OS/RIOT/images/Native.jpg)

# Hardware
- CPU: Host CPU
- RAM: Host RAM
- Flash: Host file system
- Network: Tap Interface
- UART: Runtime configurable - `/dev/tty*` are supported
- GPIO: Use the physical pins on the GPIO header of the Raspberry Pi
- SPI: Use the physical SPI port of the Raspberry Pi
- Timers: Host timer
 */
